# Spotify Clone

A web-based music player inspired by Spotify, built with HTML, CSS, and JavaScript.

## Tech Stack
- HTML
- CSS
- JavaScript

## Getting Started
```bash
git clone https://github.com/mrsandeep27/spotify.git
cd spotify
```

Open `index.html` in your browser.

## Project Structure
```
index.html      # Main player UI
style.css       # Styling
script.js       # Player logic
covers/         # Album artwork
songs/          # Audio files
logo.png        # Logo
playing.gif     # Playback animation
```

## Features
- Music playback controls
- Song list with cover art
- Animated playback indicator
- Spotify-inspired UI design
